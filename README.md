# xshot - 跨平台截图工具

**目前处在早期开发阶段，功能尚不完善，欢迎提出建议和反馈！**

xshot - 是一个跨平台的截图工具，基于 Tauri 和 React 构建，旨在提供轻量级且高效的截图体验。
